# miros

Vite-powered editorial landing page inspired by the provided EPISODES references.

## Files

- `index.html` - semantic page markup
- `styles.css` - base responsive design system, layout, and generated visual placeholders
- `src/main.js` - Vite JavaScript entry, interactions, and style imports
- `src/styles.scss` - SCSS tokens, mixins, nested refinements, and enhancements

## Preview

```bash
npm install
npm run dev
```

Use `npm run build` to create a production build.
